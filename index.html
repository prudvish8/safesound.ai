How it works

  1. Edge capture: Saifcam observes real-time video.
  2. Structured events: detections → rules → JSON with timestamps.
  3. Sign & commit: events are signed (ed25519) and written to HBUK.
  4. Optional Gemini: short-clip summaries for human-readable context.
  5. Rights enforced: export/train gates respect your selected mode.

Trust primitives

  • Per-device keys & signatures
  • Event digests + evidence hashing
  • Consent tokens & licenses per event
  • Immutable append-only storage (HBUK)
